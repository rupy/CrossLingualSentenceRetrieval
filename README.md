# Cross-lingual Sentence Retrieval

## Installation

Use "git clone" command.

## Dependency

Use "pip install" command to install dependent libraries

```
numpy 1.9.1
scikit-learn 0.15.2
scipy 0.14.1
matplotlib 1.4.2
gcca 0.1dev
```

You can download & install gcca from https://github.com/rupy/GCCA
